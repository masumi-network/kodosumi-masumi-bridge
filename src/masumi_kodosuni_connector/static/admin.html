Configuration Guide

To enable an agent:

  1. Stop the server
  2. Add the agent identifier to your .env file:
    3. AGENT_IDENTIFIER_<flow_key>=<your-agent-id>
  3. Restart the server

Example:

AGENT_IDENTIFIER_-_localhost_8001_seo-agent_-_=seo-analysis-masumi-agent

Network Options:

  • preprod - Testnet for development
  • mainnet - Production network

Note: Only agents with configured identifiers will be callable via the API.
